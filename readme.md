# nodeMCU RFID Relay Dengan Telegram Notif

Ini merupakan IoT berbasis NodeMCU v3 yang di buat untuk menyalakan suatu alat elektronik menggunakan modul relay yang harus ter autentikasi dengan modul RFID

# Alat Dan Bahan
  - Board NodeMCU v3
  - Modul RFID MFRC522
  - Modul Relay 3V
  - Kabel Jumper (F/F)
